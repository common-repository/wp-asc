== Description ==

* Author: [Anton A. Belogorodtsev](https://fe-nix.ru/about.html)
* Project URI: <https://fe-nix.ru/wp-asc.html>

When somebody copy information from your website, the plugin will add to the copied text of the link source.
When using the plug-in greatly increasing the number of backlinks.

== Installation ==

1. Upload directory `wp-asc` to the `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy!
